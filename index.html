核心功能。

为 WindSurf 用户打造的全方位账号管理工具

👤账号管理

  • 邮箱 + 密码登录 WindSurf 账号，支持批量保存
  • 一键切换已保存账号，自动写入认证文件，无需重启编辑器
  • 账号列表支持按日用量、到期时间升降序排列，排序状态持久化
  • 自动识别 Free / Trial / Pro / Teams / Enterprise 等订阅类型，各类型对应不同颜色标识

📊配额监控

  • 实时显示日已用、周已用百分比，支持 Quota 计费模式
  • 顶部显示当前账号到期时间，临期自动变色预警
  • 点击刷新按钮即时获取最新配额数据

🔄自动切号

  • 开启自动切号后，日配额使用 ≥97% 时自动切换到轮换池中下一个可用账号
  • 自由勾选参与轮换的账号，轮换池列表与账号列表共享排序规则
  • 仅切换到日已用 < 97% 的账号，所有账号用尽时发出提醒

🖥️设备管理

  • 显示当前设备的 Machine ID
  • 支持一键重置设备指纹，可设置切号时自动重置

✨界面体验

  • 各功能区块支持展开/收起，状态持久化保存
  • 底部状态栏实时显示当前账号和订阅类型
  • 配额用量按阈值显示绿/黄/红颜色指示

错误 导入账号失败

网络不可用，中转和直连均失败，通常是网络连接问题。

尝试开启代理后重新导入，或关闭所有代理重启WindSurf客户端导入，检查网络连接是否正常。
